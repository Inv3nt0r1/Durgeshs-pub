Welcome to Durgesh’s Pub

It's a media server which will let you watch Movies, Series, Educational Courses and more. Similar to Netflix you can say. You can watch any movie or show, pause it and resume it anytime. It will keep track of when you pause and will give an option to resume from that point.

It's a little effort from my side so that my family and some of my friends can watch Movies and Series without actually finding it on the internet and downloading.

The server is hosted on my Raspberry Pi 4 and it will play files that I have stored on my external Hard Drive connected to Raspberry Pi.

The server is smart enough to track any error and resolve it automatically, but still some downtimes can occur if I have any power cut or Raspberry Pi or my internet is not working. Just inform me once if you see that the server is not running.

Click on the button below and background JS will check whether the server is running or not, do some more checks and will redirect you to the server. You can login and start watching.
